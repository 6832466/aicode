# 小说推文人物改名说明

## 适用场景

- 用户说"帮我改人物名字""把角色名换掉""改一下人名"
- 文本为小说推文、分镜脚本等长篇内容，字数可能3万字以上
- 需求严格：**改角色名（保持逻辑关系），数字转汉字，核心剧情不变，字数不变，不带章节信息**

## 核心原则（必须遵守）

1. **修改所有角色名称**：将文中所有角色的名字替换为新名字
2. **名字逻辑关系保持一致**：同姓家族改后仍同姓、师徒关系反映在新名中、情侣/夫妻名字风格匹配等
3. **核心剧情不变**：故事走向、情节发展、人物关系、情感线索一字不改
4. **字数不变**：修改后文章总字数与原文基本一致，不增不减
5. **数字转汉字**：除了年月日格式的日期外，所有阿拉伯数字转为汉字
6. **不带章节信息**：输出结果中去除所有章节标记（如"第一章""第1章""Chapter 1"等）
7. **绝不修改内容**：除了角色名、数字、章节信息外，其余文字一字不动
8. **绝不添加标点**：原文没有的标点不添加，已有的标点不修改、不删除
9. **格式原样保留**：换行、空行、缩进、分隔符等格式完全照搬
10. **输入什么格式，输出什么格式**：TXT进TXT出，MD进MD出，原文的编码和扩展名不变

## 输出要求（必须遵守）

**只输出修改后的文章内容，不要其他任何东西。**

- ❌ 不输出改名对照表
- ❌ 不输出修改统计
- ❌ 不输出进度提示
- ❌ 不输出分析说明
- ❌ 不输出任何与原文无关的额外文字
- ✅ 只输出一篇完整的、修改后的文章，别无他物

## 工作流程

### 步骤1: 确认输入文件与角色信息

1. 向用户确认待处理的文件路径
2. 读取文件内容
3. **提取所有角色名称**：扫描全文，列出所有出现的人物名字
4. **判断小说题材类型**：根据内容判断题材（都市/古风/玄幻/校园/军旅/宫斗等）
5. **网上搜索常用名字**：
   - 根据判断的题材类型，在网上搜索该题材下使用较广的角色名字
   - 搜索关键词示例：`古风小说常用角色名`、`都市小说男名女名推荐`、`玄幻小说人物名字大全`、`宫斗小说妃嫔名字`
   - 优先选择该题材中出现频率高、读者接受度广的名字
   - 禁止随意编造生僻名字，新名字必须是网上能搜到的、有使用基础的名字
6. **确认新名字**：
   - 方式A：用户提供新名字对照表
   - 方式B：根据题材搜索结果，挑选合适的名字并维护逻辑关系，供用户确认后再执行
7. 统计总字数，告知用户将分几段处理

### 步骤2: 名字逻辑关系分析

在确认新名字前，必须分析并维护以下逻辑关系：

| 关系类型 | 改名规则 | 示例 |
|----------|----------|------|
| 同姓家族 | 改后仍同姓 | 原姓"王"→新姓"李"，家族成员都姓"李" |
| 夫妻关系 | 姓氏协调 | 改名后夫妻姓氏仍匹配（原夫姓=原妻随夫姓→新夫姓=新妻随新姓） |
| 师徒关系 | 名字风格呼应 | 师傅名"玄真"→徒弟名含同类字，如"玄明" |
| 兄弟姐妹 | 同辈用字规律 | 原名"王明/王辉"（同辈同偏旁）→新名保持同偏旁规律 |
| 情侣/CP | 风格匹配 | 古风配古风、现代配现代 |
| 称呼关联 | 一并修改 | "张总"改姓后→"李总"，"小明"→"小华"（新名对应小称） |
| 外号/别名 | 与新名关联 | 原外号基于原名特征→新外号基于新名特征 |

### 步骤3: 分段策略

由于文本可能超过3万字，AI无法一次性输出完毕，必须分段处理：

1. **分段规则**：每段约2000-3000字，按自然段落分割（在换行符处断开，不切断句子）
2. **段落数计算**：总字数 ÷ 2500 = 需要的段数（向上取整）
3. 向用户说明分几段处理

### 步骤4: 逐段修改

对每一段执行以下操作：

1. 读取该段原文
2. 逐句扫描，替换角色名称 + 阿拉伯数字转汉字 + 去除章节信息
3. 将修改后的该段文本追加写入输出文件
4. 简短告知用户当前进度（如"第3/12段已完成"），然后继续下一段

### 步骤5: 输出结果

1. 输出完整修改后的文件（保存到原文件同目录，文件名加 `_renamed` 后缀）
   - 例：`小说.txt` → `小说_renamed.txt`
2. 全部完成后告知用户文件已保存，仅此而已

## 新名字选取规则（必须遵守）

### 核心规则：根据题材从网上搜索常用名字

**禁止随意编造名字。** 新名字必须是网上能搜到的、在该题材下使用较广的名字。

### ⛔ 绝对禁止：不能出现明星/公众人物的名字

新名字必须通过以下排雷检查，**任何一项命中即弃用，必须换名**：

| 排雷检查项 | 说明 | 反例 |
|------------|------|------|
| 影视明星 | 演员、歌手、导演、主持人等 | 刘德华、杨幂、赵丽颖、王一博 |
| 体育明星 | 运动员、教练等 | 姚明、刘翔、谷爱凌、孙杨 |
| 网红/KOL | 大V、主播、博主等 | 李佳琦、薇娅、罗翔 |
| 历史名人 | 政治家、军事家、文学家等 | 诸葛亮、李白、苏轼、岳飞 |
| 公众人物 | 企业家、科学家等 | 马云、雷军、钟南山 |
| 小说名角色 | 其他知名小说的主角名 | 张无忌、令狐冲、萧炎、叶凡 |
| 谐音撞名 | 与明星名谐音近似的 | 刘德桦（→刘德华）、扬密（→杨幂） |

**筛选流程：选好新名字后，必须在网上搜索确认该名字不是知名人物的真实姓名或广为人知的角色名。如果有撞名，立即换掉。**

### 按题材搜索名字

| 题材类型 | 搜索关键词示例 | 名字风格 |
|----------|----------------|----------|
| 都市/现代 | `都市小说男名推荐`、`现代小说常用角色名` | 接地气、生活化：周远、李婉清、陈磊、赵静 |
| 古风/武侠 | `古风小说常用角色名`、`武侠小说人物名字大全` | 诗意文雅：慕容云、柳如烟、萧逸风、沈清歌 |
| 玄幻/仙侠 | `玄幻小说人物名字大全`、`仙侠小说角色名推荐` | 大气磅礴：叶辰、苏沐橙、楚风、林动 |
| 校园/青春 | `校园小说角色名`、`青春小说男名女名` | 清新自然：林一、苏念、江辰、陈小希 |
| 军旅/谍战 | `军旅小说人物名`、`谍战小说角色名推荐` | 刚毅厚重：周卫国、陈少杰 |
| 宫斗/宅斗 | `宫斗小说妃嫔名字`、`宅斗小说角色名推荐` | 雅致含蓄：沈眉庄、安陵容 |
| 悬疑/推理 | `悬疑小说角色名`、`推理小说人物名推荐` | 普通真实：秦风、宋义 |
| 言情/甜宠 | `言情小说男名推荐`、`甜宠小说角色名` | 苏感十足：顾南城、沈言 |

### 搜索流程

```
1. 读取全文 → 判断题材类型
2. 根据题材在网上搜索常用角色名（使用 WebSearch 工具）
3. 从搜索结果中筛选符合以下条件的名字：
   ✅ 该题材下出现频率高、读者有印象
   ✅ 符合角色性格/身份（主角名大气、配角名恰当、反派名有辨识度）
   ✅ 维护角色间逻辑关系（同姓、同辈用字等）
   ✅ 不与文中其他角色重名
   ✅ 不是明星/公众人物的真实姓名
   ✅ 不是其他知名小说的标志性角色名
   ✅ 不与明星名谐音撞名
   ❌ 不用生僻字
   ❌ 不用明显违和的名字（如古风小说出现"张伟"）
   ❌ 不用明星名字（任何领域的知名人物都不行）
4. 选好名字后，逐个在网上搜索确认不撞名
5. 将选好的新名字列表交给用户确认
6. 确认后开始逐段替换
```

### 名字匹配原则

- **主角**：选该题材下最经典、最有辨识度的名字
- **配角**：选配合主角风格、不抢主角风头的名字
- **反派**：选有压迫感或阴柔感的名字（视题材而定）
- **长辈**：选稳重、有年代感的名字
- **女性角色**：选符合角色气质的名字（温柔/泼辣/清冷/活泼等）

## 角色名称替换规则

### 必须替换的内容

| 类型 | 说明 | 示例 |
|------|------|------|
| 全名 | 角色完整姓名 | 王大明→李建国 |
| 姓氏单独出现 | 称呼、自称 | "王"→"李"（如"老王"→"老李"） |
| 名字单独出现 | 亲密称呼 | "大明"→"建国" |
| 昵称/小名 | 基于原名的小称 | "明明"→"国国"（基于新名生成） |
| 称谓 | 基于名字的称呼 | "王总"→"李总"、"王老师"→"李老师" |
| 外号/绰号 | 基于名字特征 | 需根据新名特征重新生成 |

### 不替换的内容

| 类型 | 说明 |
|------|------|
| 非角色的同名词 | 如地名中恰好含角色名的字，不替换 |
| 成语/固定搭配中的字 | 如"明明白白"中的"明"不因角色名"大明"而替换 |
| 不确定是否指代角色 | 一律保持原样，宁可漏换不错换 |

## 数字转汉字规则

### 必须转换（阿拉伯数字 → 汉字）

| 原文 | 修正 | 说明 |
|------|------|------|
| 3个人 | 三个人 | 数量词 |
| 第1次 | 第一次 | 序数词 |
| 20岁 | 二十岁 | 年龄 |
| 100块 | 一百块 | 金额（口语语境） |
| 5年前 | 五年前 | 时间跨度 |
| 3楼 | 三楼 | 楼层 |
| 2小时 | 两小时 | 时长（注意"二"与"两"的用法） |
| 18号 | 十八号 | 日期中的"号" |
| 80多 | 八十多 | 约数 |
| 3万 | 三万 | 大数（数字+万/亿等单位） |

### 不转换（保持阿拉伯数字原样）

| 原文 | 保持原样 | 原因 |
|------|----------|------|
| 2026年5月25日 | 2026年5月25日 | 年月日格式日期 |
| 1999年 | 1999年 | 年份 |
| 2025年3月 | 2025年3月 | 年月 |
| 3.14 | 3.14 | 小数（小说中罕见，但保留） |

### "二"与"两"的用法规则

- **用"二"**：序数词（第二）、号码（二号）、年份中的数词
- **用"两"**：量词前（两个人、两天）、约数（两三个）
- 常规数量根据语境自然选择，优先遵循口语习惯

## 章节信息去除规则

### 必须去除

| 类型 | 示例 |
|------|------|
| 中文数字章节 | 第一章、第十二章 |
| 阿拉伯数字章节 | 第1章、第12章 |
| 英文章节 | Chapter 1、Chapter 12 |
| 章节标题行 | 整行只有章节信息的行（包括章节号+标题名，如"第一章 初遇"） |
| 卷/部标记 | 第一卷、上部 |

### 去除方式

- 如果章节信息独占一行：**删除整行**（包括该行的换行符）
- 如果章节信息嵌入正文：仅删除章节标记部分，保留其余文字

## 不修改的内容（保持原样）

| 类型 | 说明 |
|------|------|
| 地点名称 | 城市名、建筑名等 |
| 组织/门派名称 | 除非与角色名直接绑定 |
| 物品名称 | 武器、法宝等专有名词 |
| 方言/口语表达 | 角色语言风格 |
| 网络用语 | 时代语境 |
| 古风/文学化表达 | 文风特色 |
| 不确定的内容 | 一律不改 |

## 分段处理详细规则

```
1. 读取全文 → 按换行符拆成自然段
2. 将自然段按字数合并为2000-3000字的块
3. 合并时必须在换行符处断开，绝不能切断句子中间
4. 每个块独立处理，互不影响
5. 修改后的块按顺序写入输出文件
```

## 输出文件格式

- 保存到原文件同目录，文件名加 `_renamed` 后缀
- 编码与原文一致（默认UTF-8）
- 格式与原文完全一致（换行符类型CRLF/LF照搬）
- 文件内容只有修改后的文章本身，无任何额外信息

## 常见问题处理

- **文件过大无法一次读取**：使用脚本分段读取，每次读取一部分
- **编码问题**：优先尝试UTF-8，失败则尝试GBK/GB2312
- **不确定是否角色名**：保持原文不改
- **角色名出现在成语中**：不替换成语中的字
- **同一称呼多种写法**：如"王大明/大明/老王/王总"全部按新名字体系对应替换

## 注意事项

- 宁可漏换，不可错换。不确定的绝对不改。
- 分段之间不能丢失内容，每段处理完后确认字数对齐。
- 输出文件必须与原文件格式完全一致，包括换行符类型（CRLF/LF）。
- **输出文件只包含修改后的文章，不要对照表、统计、说明等任何额外内容。**
- 改名后全文必须通顺，不能出现旧名残留或新名不一致的情况。
- **绝对不能出现明星/公众人物的名字，选好名字后必须网上搜索排雷确认。**
